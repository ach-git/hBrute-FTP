# hBrute-FTP
Simple Brute Force FTP server in python using socket library for education purpose only.

## Installation:
```
$ git clone https://github.com/ach-git/hBrute-FTP
$ cd ../hBrute-FTP
$ pip install socket

$ py Brut-Force-FTP.py
```
